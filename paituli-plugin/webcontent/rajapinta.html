Paitulin rajapinnat

Suuri osa Paitulin aineistoista on saatavilla OGC:n WMS-, WFS- ja WMTS-rajapintojen kautta. WFS rajapinta on saatavilla vain vektoriaineistoille. Tarkka lista rajapinnan kautta saatavilla olevista aineistoista löytyy tämän sivun lopusta.

Rajapinnan kautta ladatuille aineistoille pätevät samat käyttöoikeudet kuin latauspalvelun kautta ladattaessa. Aineistojen käyttöehdot ja kuvaukset löytyvät Metatiedot sivulta.

Aineistot ovat saatavilla seuraavista osoitteista:

Paituli web services

Several Paituli datasets are available as OGC web services: WMS, WFS, WMTS. WFS service is available only for vector datasets. All datasets available via web services are listed at the end of this page.

The license terms are the same for data downloaded via web services as via download service. The license terms and descriptions can be found from Metadata page.

The web services are available from these connection URLs:
